## Description

## Maker comment

Hey Product Hunt 👋

I built Founder Panel because startup advice often gets flattened into generic summaries, even when the original writing is much more specific and useful.

The project was inspired by Garry Tan's GStack and its approach to turning Claude Code into a set of focused, repeatable workflows. Founder Panel applies that idea to startup advice: `/founder-conversation` lets you think through a problem with one source-grounded founder synthesis, while `/board-room` lets 2–5 independently grounded perspectives disagree usefully.

The current release indexes 20,347 public posts from 50 founders and investors. Every substantive claim should link back to the original writing, and every session is explicit that these are AI syntheses—not the real people and not affiliated with or endorsed by them.

It is free, open source, and designed to work inside the Claude Code setup you already have. I would especially love feedback on the founder roster, citation quality, and which startup decisions you would put in front of the board first.

Thanks for checking it out.
